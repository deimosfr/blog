Jenkins : L’intégration continue

26 January 2013 in Debian, Hi-Tech, Linux, Web

Écrit en Java, Jenkins fonctionne dans un conteneur de servlets tel qu’Apache Tomcat, ou en mode autonome avec son propre serveur Web embarqué. Dans le cadre de l’amélioration de Limesurvey comme cité dans un précédent ticket, le couplage Selenium/Jenkins est très pratique. J’ai donc fais une documentation sur la mise en place de Jenkins.
