A tout cassé ? Pani pwoblem !

10 February 2012 in Hi-Tech, Linux, Red Hat
J’ai un collègue qui a fait une belle boulette (un chown -Rf mysql sur / !). Du coup la merde, il a fallu trouver une solution pour restaurer les droits. Heureusement Red Hat à penser aux têtes en l’air et a mis dans la commande rpm les options *setperms et *setugids pour réparer les droits sur les packages installés. Bref de quoi réparer votre machine. Donc si vous aussi vous avez boulétiser, sachez qu’une solution existe sur Red Hat.
