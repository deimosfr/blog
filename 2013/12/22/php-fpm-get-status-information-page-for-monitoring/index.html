PHP-FPM : get status information page for monitoring

22 December 2013 in Debian, Hi-Tech, Linux, Web

To be able to monitor the PHP-FPM service with connections, queue information, process… you’ll need to set the status page. This will give you something like this :

pool: www
process manager: dynamic
start time: 18/Dec/2013:19:00:41 +0100
start since: 52972
accepted conn: 5268
listen queue: 0
max listen queue: 0
listen queue len: 0
idle processes: 3
active processes: 1
total processes: 4
max active processes: 4
max children reached: 0

Those informations are important to correctly tune the PHP-FPM service. To add it with Nginx, you can follow my tutorial (french).
